# Seed vocabulary sets (original examples)

Original example sets showing the expected structure: level-tagged items with
collocations and a fresh example sentence each. Example sentences are written
for this plugin. For academic lists, draw on open sources (e.g. the Academic
Word List headwords, used nominatively) — never proprietary vocabulary banks.

## B2 set — expressing cause and effect

| Item | Collocations | Example |
|---|---|---|
| give rise to | give rise to concern / problems | The new schedule gave rise to complaints from commuters. |
| stem from | stem from a misunderstanding / belief | Her caution stems from an earlier failed launch. |
| bring about | bring about change / a shift | The campaign brought about a modest drop in food waste. |
| consequence | serious / unintended / direct consequence | One unintended consequence was longer queues at lunchtime. |
| trigger | trigger a debate / response / alarm | The report triggered a debate about housing costs. |

## C1 set — hedging and stance (academic writing)

| Item | Collocations | Example |
|---|---|---|
| arguably | arguably the most / an arguably flawed | This is arguably the study's weakest assumption. |
| tendency | a marked / growing tendency to | There is a marked tendency to overstate early results. |
| by no means | by no means certain / universal | Such access is by no means universal outside cities. |
| to some extent | true / valid to some extent | The model holds, to some extent, even under stress. |
| cast doubt on | cast doubt on a claim / finding | Later trials cast doubt on the original effect size. |

## C1/C2 set — precision verbs for essays

| Item | Collocations | Example |
|---|---|---|
| underpin | underpin an argument / economy | Two assumptions underpin the whole proposal. |
| offset | offset a cost / loss / risk | Higher fares were offset by shorter journey times. |
| curtail | curtail spending / freedoms | The drought forced the town to curtail water use. |
| exacerbate | exacerbate a problem / tension | Poor signage exacerbated the confusion at the exits. |
| mitigate | mitigate the impact / risk | Tree cover mitigates the heat-island effect. |

## Spaced-repetition usage

Store review state per item in `<base>/vocab-box.json` (Leitner boxes 1–5
with review intervals next session → 1 day → 3 days → 7 days → 30 days
(mature); this must match `vocabulary-builder/SKILL.md`). New items enter box
1, due next session. Quiz items due today; a correct recall moves the item up
one box, a miss returns it to box 1.
